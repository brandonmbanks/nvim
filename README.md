# My Neovim Config

## Try out this config
Make sure to remove or move your current nvim directory

```sh
git clone git@github.com:brandonbanks/nvim.git ~/.config/nvim
```

Run `nvim` and wait for the plugins to be installed

## Check health

Open `nvim` and enter the following:

```
:checkhealth
```
